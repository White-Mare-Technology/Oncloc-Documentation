# Project Layout

## Structure
- `mkdocs.yml`: The configuration file for your MkDocs project.
- `docs/`: The documentation directory containing all your Markdown files.
  - `index.md`: The documentation homepage.
  - `dashboard/`: Contains documentation related to the Dashboard/Portal.
  - `mobile/`: Contains documentation related to the Mobile Application.

## Example
```plaintext
mkdocs.yml    # The configuration file.
docs/
    index.md  # The documentation homepage.
    dashboard/
        how_to_navigate.md
        project_layout.md
    mobile/
        introduction.md
        installation.md
        features.md
        user_guide.md
